## 🧰 Development

```bash
# Clone the repository
git clone https://github.com/konnn04/konnn04.github.io.git

# Navigate to project folder
cd konnn04.github.io

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
